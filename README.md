## 🚀 Features

### 💸 Expense & Income Management

* Add and manage transactions easily
* Categorize expenses (Food, Travel, Bills, etc.)
* Add custom notes (EB Bill, Shopping, etc.)
* Real-time balance updates

---

### 📊 Analytics & Visualization

* Line Chart 📈 (weekly trends)
* Bar Chart 📊 (category comparison)
* Pie Chart 🥧 (distribution)
* Smart insights and warnings

---

### 🎯 Budget Management

* Set monthly budget
* Track usage with progress bar
* Alerts when limit exceeds

---

### 👥 Friends & Transfers

* Add friends
* Send/receive money
* Track:

  * You owe
  * Owed to you

---

### 🔔 Reminders

* Bill reminders (Rent, EB, etc.)
* Notification toggle

---

### 📈 Smart Prediction

* Predict future expenses
* Analyze spending behavior

---

### ⚙️ Settings & Customization

* Dark mode 🌙
* Currency selection
* Profile management
* Export report (PDF)

---

## 🎨 UI/UX Highlights

* Premium dark theme 🌙
* Smooth iOS-like animations 🎬
* Floating action button ➕
* Swipe navigation 🔄
* Clean card-based design 💳

## 🚀 Run Locally

```
git clone https://github.com/Sanjay1771/expense-tracker-app.git
cd expense-tracker-app
flutter pub get
flutter run
```

---

## 📦 Build APK

```
flutter build apk
```

Output:

```
build/app/outputs/flutter-apk/
```
